📌 Overview
This repository demonstrates data cleaning techniques applied to a healthcare dataset.
The notebook walks through handling missing values, duplicates, and incorrect data types, ensuring the dataset is ready for analysis and modeling.

🎯 Objectives
• 	Identify and handle missing values (numeric & categorical).
• 	Detect and remove duplicate records.
• 	Correct data type inconsistencies.
• 	Provide a traceable, step-by-step cleaning workflow.

📂 Repository Structure
data_cleaning/
├── Healthcare_Data_Cleaning.ipynb   # Main notebook
└── README.md                        # Documentation

📊 Workflow
1. 	Load Dataset → Import healthcare data.
2. 	Explore Data → Check shape, info, and summary statistics.
3. 	Handle Missing Values → Imputation strategies (mean, median, mode).
4. 	Remove Duplicates → Drop duplicate rows.
5. 	Fix Data Types → Convert columns to appropriate types.
6. 	Validate Results → Compare before vs. after cleaning.
